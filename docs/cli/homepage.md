# govex homepage

Generate a Markdown homepage for vulnerability report websites.

## Usage

```bash
govex homepage [flags]
```

## Description

The `homepage` command creates a Markdown index page for vulnerability tracking websites. This is useful for creating documentation sites that aggregate vulnerability reports across multiple projects or repositories.

## Use Cases

- Generate landing pages for vulnerability documentation sites
- Create index files for MkDocs or similar static site generators
- Build vulnerability dashboards in git-based workflows
