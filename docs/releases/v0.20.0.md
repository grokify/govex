# v0.20.0 Release Notes

**Release Date:** June 28, 2026

## Highlights

This release introduces **pentest remediation status reports**, enabling teams to generate professional remediation status letters that track penetration test findings, compensating controls, and remediation milestones.

## New Features

### Pentest Remediation Status Reports

The new `reports/pentest` package provides types and utilities for generating pentest remediation status letters:

#### Report Structure

- **Metadata** - Assessment details including assessor, dates, scope
- **Findings Summary** - Counts by severity (Critical, High, Medium, Low)
- **Remediation Status** - Total/Fixed/Pending with percentage complete
- **Pending Items** - Findings still awaiting remediation with owner and target dates
- **Compensating Controls** - Mitigations in place for pending items
- **Milestones** - Upcoming remediation phases with target dates

#### Output Formats

- **JSON** - Structured data with full JSON Schema support
- **Markdown** - Formatted tables for documentation
- **PDF** - Professional reports via maroto library

### Example Usage

```go
import "github.com/grokify/govex/reports/pentest"

// Create a new report
report := pentest.NewReport("Q2 2026 Pentest Remediation Status")
report.Metadata = pentest.Metadata{
    ReportName:     "External Network Penetration Test - Q2 2026",
    Assessor:       "SecureCo Consulting",
    AssessmentDate: "2026-04-15",
    ReportDate:     "2026-06-15",
    Author:         "Security Team",
    Scope:          "External-facing web applications and APIs",
}

// Set findings counts
report.Findings = pentest.FindingsSummary{
    Critical: 2,
    High:     5,
    Medium:   8,
    Low:      12,
}

// Add pending items
report.PendingItems = []pentest.PendingItem{
    {
        ID:         "PENTEST-001",
        Title:      "SQL Injection in Search API",
        Severity:   "Critical",
        Status:     "In Progress",
        Owner:      "Platform Team",
        TargetDate: "2026-07-01",
    },
}

// Calculate status and generate output
report.CalculateStatus()
report.WriteMarkdownFile("remediation_status.md")
report.WriteJSONFile("remediation_status.json")
```

### JSON Schema for AI Integration

Generate JSON Schema for AI agent integration and validation:

```go
schema, err := pentest.JSONSchemaString()
```
